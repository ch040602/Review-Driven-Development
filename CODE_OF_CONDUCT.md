# Code of Conduct

This project expects professional, respectful collaboration.

## Expected behavior

- Keep critique technical and evidence-based.
- Separate disagreement about design from personal criticism.
- Provide reproduction steps for bug reports.
- Respect maintainers' decisions on scope and safety.

## Unacceptable behavior

- Harassment, threats, or personal attacks
- Publishing secrets or private information
- Intentionally weakening safety, validation, or review gates without disclosure

Maintainers may close issues or PRs that violate these expectations.
